# CZN 한글 위키 사이트 디자인

## 목적
- 한글 CZN 위키가 부재한 상황에서, 직접 수집/정리한 데이터를 위키 사이트로 제작
- 슈퍼크리에이티브 포트폴리오 제출용 (커뮤니티 공개 가능한 품질)
- 게임에 대한 깊은 이해 + 데이터 파이프라인 구축 역량 증명

## 프로젝트 구조 변경
- 기존 전투계산기/덱빌더 → 레거시 (`/deck-builder`, 네비 미노출)
- 위키 페이지 → 메인 컨텐츠
- 분석서(Marp 슬라이드) → 병렬 진행, 완성 후 전투 시스템 페이지에 반영

## 디자인 방향
- Prydwen 스타일 카드형 레이아웃
- 현재 다크 테마 + 속성별 컬러 시스템 유지
- 모바일 대응 (mobile-first, md: 768px)

## 아키텍처: App Router + SSG

```
src/app/
  (wiki)/                    # 위키 레이아웃 그룹
    layout.tsx               # 탑 네비 + 푸터 (캐릭터/카드/용어집/전투시스템)
    page.tsx                 # 홈 (빈 껍데기, 통계 없음)
    characters/
      page.tsx               # 캐릭터 목록 (28명, 카드형 그리드 + 필터)
      [id]/page.tsx          # 캐릭터 상세 (스탯/카드/바리에이션)
    cards/
      page.tsx               # 전체 카드 DB (322장, 필터/검색)
      [id]/page.tsx          # 카드 상세
    glossary/
      page.tsx               # 용어집 (상태효과/메커니즘/키워드)
    combat/
      page.tsx               # 전투 시스템 (공백, 분석서 완성 후)
  deck-builder/
    page.tsx                 # 레거시 (네비 미노출, 기존 코드 이동)
```

- `generateStaticParams`로 28캐릭터 + 322카드 빌드 타임 정적 생성
- JSON 데이터가 빌드 타임에 확정 → SSG와 완벽 호환, Vercel 무료 티어 최적

```
┌─────────────────────────────────────────────────┐
│  [로고/CZN]   캐릭터  카드  용어집  전투시스템   │  탑 네비
├─────────────────────────────────────────────────┤
│              페이지 컨텐츠                        │
├─────────────────────────────────────────────────┤
│  (c) 2026 CZN Wiki · 비공식 팬 사이트            │  푸터

```

- 홈은 로고 클릭으로 접근, 네비에 별도 탭 없음
- 모바일: 햄버거 메뉴
- 현재 페이지 하이라이트

## 캐릭터 페이지

### 목록 (`/characters`)
- 카드형 그리드: 4열(데스크탑) / 2열(모바일)
- 필터: 속성, 클래스, 진영, 검색
- 카드에 표시: 이미지, 이름(한글), 레어리티(별), 클래스, 속성 배지

### 상세 (`/characters/[id]`)
- 상단: 캐릭터 프로필 + Lv60 스탯 (ATK/DEF/HP)
- 기본 카드 4장: prydwen 스타일 카드 컴포넌트
- 고유 카드 4장: 각 카드별 기본 + #1~#5 바리에이션 종속 표시

고유 카드 배치:
```
┌─ 카드 A ──────────────────────────────────┐
│  기본: [AP2 공격] 효과 텍스트              │
│  ├ #1: [AP2 공격] 변경된 효과              │
│  ├ #2: [AP1 스킬] 변경된 효과              │
│  ├ #3: [AP2 강화] 변경된 효과              │
│  ├ #4: [AP1 공격] 변경된 효과              │
│  └ #5: [AP3 스킬] 변경된 효과              │
└───────────────────────────────────────────┘
```

## 카드 컴포넌트 (Prydwen 레이아웃 재현)

```
┌──────────────────────────┐
│ [AP] 카드명               │  상단: 속성색 좌측 보더
│      타입아이콘 Attack     │  + AP 배지(원형) + 카드명 + 타입
│                          │
│    [카드 일러스트 배경]      │  중앙: 200x300 아트워크
│                          │  (prydwen CDN 이미지)
│                          │
│    ──────────────────    │
│    100% 피해.             │  하단: 효과 텍스트 (한글)
│    [소멸] [보존]          │  + 태그 배지
└──────────────────────────┘
```

- 이미지: prydwen.gg `/static/` CDN (200x300 webp, 일러스트만)
- 파일명 패턴: `{charId}_{번호}.webp` (예: `ronoa_1.webp`)
- 속성별 테두리 색상: Void=보라, Order=파랑, Justice=금색, Instinct=초록, Passion=빨강
- 타입별 아이콘 색상: Attack=빨강, Skill=파랑, Upgrade=노랑

## 카드 페이지 (`/cards`)
- 캐릭터 카드 200장 + 중립 122장 = 322장 통합 DB
- 필터: 타입, 태그, AP 코스트, 카테고리(기본/고유/중립/몬스터)
- effectKeywords 기반 키워드 필터 (사기, 취약, 드로우 등)
- 카드 클릭 → `/cards/[id]` 상세 (효과, 바리에이션, 소속 캐릭터 링크)

## 용어집 (`/glossary`)
- glossary.json 기반
- 탭: 상태 효과 | 메커니즘 | 카드 키워드
- 상태 효과: buff/debuff 그룹핑
- 각 항목 펼침: 중첩 여부, 트리거, 최대 중첩, 감소 조건, 한영 설명

## 전투 시스템 (`/combat`)
- 현재 공백. 분석서(combat-mechanics.md) 완성 후 웹 가이드로 변환 예정
