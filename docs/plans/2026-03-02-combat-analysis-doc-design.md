# 전투 매커니즘 분석서 — 디자인 문서

## 개요
- **목적**: 슈퍼크리에이티브 개발QA 지원 포트폴리오 필수 제출 서류
- **독자**: 슈퍼크리에이티브 개발팀
- **포맷**: Marp (Markdown → PPT/PDF), 다크 테마
- **접근법**: 시스템 레이어 (Top-Down) — 전체 시스템 → 카테고리 → 개별 태그
- **데이터**: JSON 데이터 기반 + 플레이 경험 보충
- **예상 분량**: ~30슬라이드

## 문서 구조

### 섹션 1: 표지 + 개요 (3장)
- 슬라이드 1: 표지 (제목, 작성자, 작성일)
- 슬라이드 2: 분석 목적 & 범위 (28캐릭터, 200장 카드, 16개 태그)
- 슬라이드 3: 전투 시스템 아키텍처 다이어그램

### 섹션 2: 전투 시스템 기초 (4장)
- 슬라이드 4: 턴 구조 & AP 시스템
- 슬라이드 5: 카드 시스템 (Starting vs Epiphany, 타입, 바리에이션)
- 슬라이드 6: 데미지 공식 (ATK-based, DEF-based, 크리티컬 기대값)
- 슬라이드 7: 효과 타입 분류 (damage, buff, debuff, shield, heal, draw, create, discard)

### 섹션 3: 태그 카테고리 맵 (1장)
- 슬라이드 8: 16개 태그를 5개 기능별 카테고리로 분류
  - 공격 강화: 점화, 분쇄, 약점 공격
  - 타이밍/조건: 개전, 종극, 신속, 주도
  - 자원 관리: 회수, 소멸, 증발
  - 방어/유지: 보존, 천상
  - 특수: 유일, 탄환

### 섹션 4: 태그별 상세 매커니즘 (~20장)
각 태그 통일된 구조:
- 발동 조건
- 효과
- 등장 카드 수 및 캐릭터별 분포
- 주요 카드 예시 (카드명 + 수치)
- 전략적 의미 (덱 빌딩/전투 흐름에서의 역할)
- 차트 또는 카드 이미지

순서: 공격 강화 → 타이밍/조건 → 자원 관리 → 방어/유지 → 특수

### 섹션 5: 태그 간 시너지 & 상호작용 (2~3장)
- 태그 시너지 매트릭스 (강력한 조합, 상충 조합)
- 캐릭터별 태그 프로필
- 태그 분포 히트맵 (캐릭터 × 태그)

### 섹션 6: 마무리 (1장)
- 분석 요약 & QA 시사점
- 핵심 발견 사항
- QA 관점 주의 매커니즘

## 기술 스택
- Marp CLI로 Markdown → PPTX/PDF 변환
- 프로젝트 내 `docs/analysis/` 디렉토리에 Marp 파일 배치
- 카드 데이터 통계는 `src/data/cards.json`에서 추출

## 태그 데이터 현황
- 16개 고유 태그: 개전, 보존, 분쇄, 소멸, 소멸 2, 신속, 약점 공격, 유일, 점화, 종극, 주도, 증발, 천상, 탄환, 회수, 회수 4
- 200장 카드 (Starting + Epiphany)
- 80장에 바리에이션 포함
