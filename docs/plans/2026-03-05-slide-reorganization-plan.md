# 슬라이드 재구성 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** combat-mechanics.md 슬라이드를 탑다운 구조로 재배치하여 논리적 흐름 확보

**Architecture:** 단일 Marp 마크다운 파일(`docs/analysis/combat-mechanics.md`)의 `---` 구분 슬라이드 블록을 설계안 순서로 재조립. 기존 콘텐츠 삭제 없이 재배치 + 섹션 타이틀 4개 신규 추가.

**Tech Stack:** Marp CLI (서버 모드 http://localhost:8080 에서 실시간 확인)

**설계 문서:** `docs/plans/2026-03-05-slide-reorganization-design.md`

---

## 용어

- **S{N}**: 현재 슬라이드 번호 (1부터 시작, `---` 구분자 기준)
- 각 태스크는 파일 전체를 Write 도구로 재작성하는 방식 — Edit으로 블록 이동은 위험

## 사전 준비

Marp 서버가 http://localhost:8080/combat-mechanics.md 에서 실행 중인지 확인.
실행 중이 아니면: `npx @marp-team/marp-cli --server docs/analysis/ --html --allow-local-files --theme docs/analysis/czn-dark.css`
