# 전투 매커니즘 분석서 — 슬라이드 재구성 설계

**날짜**: 2026-03-05
**목적**: 탑다운 구조로 재배치하여 논리적 흐름 확보
**대상 독자**: 개발팀 (설계/밸런스)
**원칙**: 기존 슬라이드 삭제 없이 재배치 + 섹션 타이틀 4개 신규 추가

## 핵심 변경 3가지

1. **턴 흐름을 6번째 슬라이드로** — 파티 구성 직후, 모든 매커니즘의 뼈대를 먼저 제시
2. **데미지 공식을 카드 직후로** — 카드 타입·효과를 배운 바로 다음에 연산 방법
3. **번뜩임 통합** — 표 설명 + 스크린샷을 연속 배치, 맥락 끊김 해소
